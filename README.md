# mudlet-distrobox

This image is intended for use with Distrobox.

## Install Distrobox

```shell
gh repo clone 89luca89/distrobox && bash install
```

## Setup

```shell
distrobox create -n Mudlet -i docker.io/gesslardev/mudlet-distrobox --yes
distrobox enter Mudlet -- distrobox-export -a mudlet
```
